If user has 0 active listings → return 200 with empty suggestions array and message: "List some items to get AI suggestions" Step 2 — Build Claude prompt Construct this system prompt: "You are a smart clothing swap assistant. Your job is to look at someone's wardrobe and identify which of their items would make the best swap offer for a specific item they want. Consider: estimated value match, size compatibility, category relevance, condition, and overall swap appeal. Always respond with valid JSON only — no markdown, no explanation outside the JSON." Construct this user message: "I want to swap for this item: - Title: {viewedItem.title} - Category: {viewedItem.category} - Size: {viewedItem.size} - Condition: {viewedItem.condition} - Estimated value: £{viewedItem.estimatedValue} Here are my active listings I could offer in exchange: {userListings.map((item, i) => ` ${i + 1}. ID: ${item._id} Title: ${item.title} Category: ${item.category} Size: ${item.size} Condition: ${item.condition} Estimated value: £${item.estimatedValue}`).join('\n')} Return a JSON object with this exact shape: { "suggestions": [ { "itemId": "<_id of the item>", "title": "", "reason": "" } ] } Return the top 3 items only. If fewer than 3 items exist, return all of them. Respond with raw JSON only."
